Submit a Boundary

Fill out this form to make a new boundary submissions to geoBoundaries. The information you provide will be formatted, packaged, and submitted as a new pull request to the geoBoundaries Github repository. After submission, we will manually assess your provided files and information, and as warranted follow up for any clarifications. Please note that all submissions must be compatible with the CC-BY SA license, which allows for most uses so long as attribution is provided. For more information, please review the Guidelines for Submission.

Drag-and-drop or select a zipped shapefile. Data must use WGS84 projection (EPSG 4326). Please note that if your submission fails, you may be required to re-upload this zip file.
