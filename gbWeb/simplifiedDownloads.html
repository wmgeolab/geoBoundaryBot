Simplified Country Files

On this page, you'll find download links for the geoBoundaries simplified dataset, which is less accurate, but much faster rendering. The geoBoundaries open license (CC-BY 4.0) country dataset seeks to represent every nation "as they would represent themselves", with no special identification of disputed areas. If you would prefer data that explicitly includes disputed areas, please see the global composite dataset. Attribution is required for all uses of this dataset.
